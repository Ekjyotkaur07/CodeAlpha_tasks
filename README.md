# ✅ To-Do List Application
## CodeAlpha C++ Internship Project

### 🎯 Project Overview
A professional console-based task management system that allows users to efficiently manage their daily tasks with features like task tracking, file persistence, and color-coded interface.

### ✨ Features Implemented

| Feature | Status | Description |
|---------|--------|-------------|
| Add Tasks | ✅ | Create new tasks with unique IDs |
| View Tasks | ✅ | Display all tasks with color coding |
| Mark Complete | ✅ | Mark tasks as completed |
| Delete Tasks | ✅ | Remove tasks from list |
| File Persistence | ✅ | Auto-save and load tasks from file |
| Color-coded UI | ✅ | Visual feedback (Green=Success/Completed, Red=Pending, Cyan=Menu) |

### 🛠️ Technologies Used
- **C++17** - Core programming language
- **STL Vector** - Dynamic task storage
- **File Streams (fstream)** - Data persistence
- **Windows API** - Console color formatting

g++ main.cpp -o taskmanager.exe
./taskmanager.exe
